Seletores CSS

Bem-vindo!

No CSS nós possuímos os seletores por ID

  • Seletor por Tipo: Busca elemento por tag HTML especifica
  • Seletor por ID(#): Busca elemento através do atributo "id"
  • Seletor por Classe(.): Busca elemento através do atributo "classe"
  • Seletor Universal(*): seleciona todos os elementos HTML
  • Seletor de Atributo([atrib]):Seleciona elementos que pussuem um atributo especifico em nosso documento HTML e conseguimos também buscar atributos com um valor especifico
